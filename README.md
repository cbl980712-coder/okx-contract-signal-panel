# OKX 合约信号面板 v1.7

> 多周期结构分析 · 自动信号扫描 · 模拟自动交易系统  
> 数据来源：OKX 公开行情接口，无需 API Key

---

## 功能概览

### 📊 自动信号扫描
- 每次扫描 60~70 个主流 USDT 永续合约
- **三层周期架构**：4h 大方向偏置 → 1h 定结构 → 15m/1h 触发
- 信号分三档：**A+（最优）/ A/B（可执行）/ C（观察）**
- 技术指标：EMA20/60、RSI、ATR

### 🎯 执行参考卡片
- **进场区间**：基于 1h 结构支撑/阻力，回踩等待，不追涨
- **止损价**：confirmed pivot low/high + buffer，结构外止损
- **止盈1/2/3**：来自真实结构高低点，禁止 ATR 推算
- **盈亏比**：≥ 1.1 才展示执行建议，废单自动过滤
- **执行建议**：主交易单 / 轻仓试单 / 仅观察

### 💰 收益预估计算器
- 滑块调节杠杆倍数（1x~100x）
- 选择止盈目标（TP1/TP2/TP3）
- 实时计算盈利/亏损 USDT 金额

### 🤖 模拟自动交易系统
- 信号自动在模拟账户开仓，还原交易所场景
- 含手续费（0.05%）、滑点（0.03%）、杠杆、爆仓机制
- 分批止盈（TP1 到了先平一半，剩余继续持有）
- 保本上移止损
- 初始资金 10,000 USDT，每单保证金 100 USDT，最多 30 仓
- 实时浮盈计算、历史平仓记录、胜率统计、最大回撤

### 📈 信号详情页
- 趋势方向、RSI、EMA 状态
- 进场区间可视化
- 持仓建议文字说明
- 信号存活时间追踪

## 技术实现

- **数据来源**：OKX 永续合约公开行情（`/api/v5/market/tickers`、`/api/v5/market/candles` 等）
- **结构分析**：confirmed pivot high/low、swing 高低点识别
- **止损逻辑**：结构外 + `max(0.5×ATR, min_sl_pct×entry)` 缓冲
- **TP 逻辑**：优先 1h confirmed pivot swing 高低点，RR≥1.0 才有效
- **RR 分层**：≥1.5 → 主交易单；≥1.1 → 轻仓试单；其余仅观察
- **失效保护**：30分钟绝对存活，4小时后才进入失效判断
- **OKX 专属**：invalid_buffer ×1.5，expire_bars ×1.5+2
